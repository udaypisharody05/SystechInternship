# Systech Solutions Internship

This repository contains the deliverables completed during my Systech Solutions internship, focused on PostgreSQL, database design, automation, and data engineering workflows.

## Deliverables

### Deliverable 1: Star Schema Design and Data Loading
- Designed dimension and fact tables using PostgreSQL.
- Created customer, product, date, and sales tables.
- Inserted sample data into the schema.
- Generated an ER diagram to visualize table relationships.

### Deliverable 2: Python-Based Database Automation
- Automated table creation using Python.
- Automated data insertion from SQL scripts.
- Connected Python to PostgreSQL using psycopg2.
- Improved repeatability of database setup.
